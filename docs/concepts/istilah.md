# Istilah

[← Kembali ke Concepts](./README.md)

Glosarium ini menjelaskan istilah teknis yang digunakan pada dokumentasi.

| Istilah | Penjelasan singkat |
|---------|--------------------|
| Enum | Daftar nilai tetap yang sudah ditentukan sehingga aplikasi tidak menerima nilai sembarangan. |
| Backed Enum | Enum yang setiap nilainya memiliki kode string atau angka yang tetap. |
| Attribute | Metadata yang ditempelkan langsung pada bagian kode yang dijelaskannya. |
| Reflection | Kemampuan aplikasi membaca struktur dan metadata kode saat aplikasi berjalan. |
| DTO | Object sederhana untuk membawa data yang sudah siap digunakan. |
| HTTP status | Angka standar pada response API yang menjelaskan hasil sebuah permintaan. |
| Category | Kelompok penyebab atau konteks error. |
| Severity | Tingkat dampak error terhadap pengguna, proses, atau layanan. |
| Runtime | Waktu ketika aplikasi sedang berjalan dan melayani proses. |
| Registry | Tempat terpusat untuk mengumpulkan dan mengelola informasi dari banyak aplikasi. |
| Consumer | Komponen yang menggunakan hasil atau data dari komponen lain. |
