# 1.3 ErrorDefinitionReader

## Tujuan

`ErrorDefinitionReader` bertugas membaca metadata error dari sebuah enum case, lalu menggabungkan kode error dari enum dengan metadata dari `#[ErrorDefinition]` menjadi object yang siap digunakan oleh consumer.

- enum sebagai sumber kode error;
